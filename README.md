# Pixel DeskPet

Pixel DeskPet 是一个 Windows 桌面像素宠物应用。它不是一个静态悬浮贴图，而是一个可以陪伴工作、提醒休息、响应 Codex 状态的轻量桌面伙伴。

项目使用 PyQt5 单文件实现，启动成本低，不依赖后端服务。宠物会常驻桌面、透明置顶、支持拖拽和右键交互，并通过本地消息文件与 Codex 或其他脚本联动。

## 产品定位

Pixel DeskPet 面向长期在电脑前工作的人，尤其适合把它放在桌面角落作为轻量陪伴工具：

- 工作时用像素宠物降低工具感和枯燥感
- 通过喝水、休息、专注提醒改善久坐习惯
- 通过 Codex 联动，把 AI 助手的状态转化为可见反馈
- 用低干扰的桌面气泡提示任务进度、完成、等待确认或错误

## 功能亮点

### 多宠物像素外观

内置 5 种宠物，每种都有独立配色、轮廓和气质：

- 奶茶猫
- 焦糖柴犬
- 赤焰狐
- 月光兔
- 薄荷史莱姆

宠物不是简单换色，绘制层包含高光、暗部、斑纹、表情、尾巴动作和物种差异。

### 配饰系统

右键菜单可切换配饰：

- 无配饰
- 小围巾
- 蝴蝶结
- 小王冠

### 桌面互动

支持常用桌宠交互：

- 拖拽移动位置
- 双击摸头
- 右键喂食、玩耍、睡觉
- 查看心情、精力、饱腹状态
- 恢复状态
- 自动散步

### 工作提醒

内置轻量提醒：

- 喝水提醒
- 休息提醒
- 勿扰模式
- 状态条开关
- 动画速度调节

### 专注陪伴

右键菜单可以开启专注陪伴：

- 25 分钟
- 45 分钟

专注期间宠物会保持专注状态，不会被随机行为打断，结束后提醒你休息。

## Codex 联动

Pixel DeskPet 支持通过本地消息文件接收 Codex 状态事件。Codex 或其他脚本可以把“正在思考、正在处理、完成、等待确认、错误”等状态推给桌宠，桌宠会自动切换情绪并弹出气泡。

支持事件：

| 事件 | 含义 |
| --- | --- |
| `thinking` | Codex 正在思考，会轮播等待话术 |
| `working` | Codex 正在处理任务 |
| `done` | 任务完成，宠物开心报喜 |
| `waiting` | 等待用户确认下一步 |
| `error` | 遇到问题，需要查看错误信息 |

示例：

```bat
Codex状态.bat thinking
Codex状态.bat working
Codex状态.bat done
Codex状态.bat waiting
Codex状态.bat error
```

自定义完成提示：

```bat
Codex状态.bat done "哇！Codex 帮你搞定刚刚提出的需求了，请查收！"
```

普通气泡消息：

```bat
宠物说话.bat "今天也要记得喝水。"
```

也可以直接调用 Python：

```powershell
python deskpet.py --say "哇！Codex 正在和你的小宠物联动。" --mood happy --duration 8000
python deskpet.py --event thinking
python deskpet.py --event done "需求已经处理好，请验收。"
```

## 安装与运行

### 环境要求

- Windows
- Python 3.11
- PyQt5

本项目当前启动脚本使用本机 Python 路径：

```text
C:\Users\86130\AppData\Local\Programs\Python\Python311\python.exe
```

如果你的 Python 安装路径不同，需要修改 `.bat` 文件里的解释器路径。

### 启动桌宠

双击：

```bat
启动小宠物.bat
```

或者运行：

```bat
run.bat
```

## 右键菜单

桌宠支持右键菜单：

- 选择宠物
- 切换配饰
- 查看当前状态
- 摸头、喂食、玩耍、睡觉
- 标记“我喝水了”
- 开启专注陪伴
- Codex 联动演示
- 恢复状态
- 设置
- 退出

## 设置项

设置窗口支持：

- 宠物大小
- 透明度
- 喝水提醒间隔
- 休息提醒间隔
- 是否启用提醒
- 是否显示状态条
- 是否允许自动散步
- 勿扰模式
- 动画速度
- 重置位置

## 项目结构

```text
deskpet/
  deskpet.py              主程序
  README.md               项目说明
  .gitignore              Git 忽略规则
  run.bat                 控制台启动脚本
  启动小宠物.bat            无控制台启动脚本
  宠物说话.bat              发送普通气泡消息
  Codex状态.bat            发送 Codex 状态事件
  诊断.bat                 诊断启动脚本
```

运行时会生成本地状态文件，这些文件不会提交到 Git：

```text
deskpet_settings.json     用户设置和窗口位置
deskpet_message.json      外部消息通道
__pycache__/              Python 缓存
```

## 技术实现

- PyQt5 创建透明、无边框、置顶桌面窗口
- QPainter 绘制像素风宠物
- QTimer 驱动动画、提醒、外部消息监听
- JSON 文件持久化设置
- JSON 文件作为本地消息通道
- 右键菜单提供全部交互入口
