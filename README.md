# Pixel DeskPet

一个 PyQt5 像素风桌面宠物，支持多宠物选择、配饰、状态互动、喝水/休息提醒，以及 Codex 联动气泡。

## 启动

双击 `启动小宠物.bat` 启动桌宠。

## 互动命令

发送普通消息：

```bat
宠物说话.bat "哇！Codex 帮你搞定刚刚提出的需求了，请查收！"
```

发送 Codex 状态事件：

```bat
Codex状态.bat thinking
Codex状态.bat working
Codex状态.bat done
Codex状态.bat waiting
Codex状态.bat error
```

## 主要功能

- 5 种像素宠物：奶茶猫、焦糖柴犬、赤焰狐、月光兔、薄荷史莱姆
- 配饰：无配饰、小围巾、蝴蝶结、小王冠
- 桌面拖拽、透明置顶、状态条、自动散步
- 摸头、喂食、玩耍、睡觉、专注陪伴
- 喝水和休息提醒
- Codex 联动状态：思考中、处理中、完成、等待确认、错误提示
